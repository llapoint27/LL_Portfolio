Liri Bot

Node command line app.

LIRI is like iPhone's SIRI. However, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface. LIRI will be a command line node app that takes in parameters and gives you back data. LIRI bot will run four functions when prompted. You will be able to search for a song within the Spotify API ('spotify-this-song), for movies on OMBD API ('movie-this') and for concerts using the Bands In Town API.
